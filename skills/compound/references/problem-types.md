# Problem Types

Problem type classifications used by the compound skill.

## Type List

Code structure, design patterns, modularization

```yaml
problem_type: architecture
```

**Examples:**
- Component separation approaches
- Directory structure decisions
- Layer dependencies

## Classification Guide

1. Select the type that matches the **core problem**
2. If multiple types apply, choose based on the **primary learning point**
3. If uncertain, use `other` and supplement with tags
